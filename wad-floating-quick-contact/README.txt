Adjustable floating The GreenChat button for your website.

== Description ==

With this plugin you can have a GreenChat floating button on your bottom right or left of your website. It has necessary options to configure the style of your button and the default message to send.

== Support the Development ==

If you like this plugin and want to support me maintaining and improving it, please send me any amount of donation to my PayPal account: https://paypal.me/habibieamrullah

= Languages =

Right now it is only in English. More languages come soon.

== Installation ==

To use this plugin, after being activated, you can see the settings page on your Dashboard to configure your GreenChat button.
